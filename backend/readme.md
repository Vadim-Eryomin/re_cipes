# Сервер для мобильного приложения ReCipes

Подготовка
``` pip install -r requirements.txt ```

Запуск 
``` python main.py ```

## Ключевые особенности: 
- Данные хранятся в social_network.db (сервер БД не нужен)
- Файлы хранятся в ./uploads/
- Сервер доступен по адресу: ``` http://127.0.0.1:5000 ```
- СЕРВЕР РАБОТАЕТ В РЕЖИМЕ РАЗРАБОТКИ
